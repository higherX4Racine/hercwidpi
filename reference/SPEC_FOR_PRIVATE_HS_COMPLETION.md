# Column specification for data about Completion at Private High Schools

Column specification for data about Completion at Private High Schools

## Usage

``` r
SPEC_FOR_PRIVATE_HS_COMPLETION
```

## Format

### SPEC_FOR_PRIVATE_HS_COMPLETION

A list of 5 elements

- COUNTY_CODE:

  `<chr>`

- CITY:

  `<chr>`

- CHOICE_IND:

  `<chr>`

- LOW_GRADE:

  `<chr>`

- HIGH_GRADE:

  `<chr>`
