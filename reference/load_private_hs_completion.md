# Syntactic sugar for loading reports about completing private high school

Syntactic sugar for loading reports about completing private high school

## Usage

``` r
load_private_hs_completion(.files)
```

## Arguments

- .files:

  paths to csv files that contain data from WISEDash public

## Value

a data frame with 17 columns, as described by the specs in "See Also"
