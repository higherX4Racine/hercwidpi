# Syntactic sugar for loading reports about enrollment after high school completion.

Syntactic sugar for loading reports about enrollment after high school
completion.

## Usage

``` r
load_ps_enrollment(.files)
```

## Arguments

- .files:

  paths to csv files that contain data from WISEDash public

## Value

a data frame with 20 columns, as described by
[SPEC_FOR_PS_ENROLLMENT](https://higherx4racine.github.io/hercwidpi/reference/SPEC_FOR_PS_ENROLLMENT.md)
